# Collections of C Projects 💻

> **⚠️ LEGACY REPOSITORY:** This repository contains a collection of older C programming projects, practices, and exercises completed in the past. It serves as an archive of my earlier work and learning progress. It is no longer actively maintained or updated.

## 📖 About
This repository holds a variety of C language practices and custom library implementations. It showcases foundational programming skills, algorithmic exercises, and standard library recreations. 

## 📂 Project List
Here is a list of the folders and projects included in this collection:

* **Advance:** Contains more complex C exercises and advanced programming concepts.
* **Basics:** A collection of fundamental C programming exercises and beginner-level scripts.
* **ft_printf:** A custom implementation of the standard C `printf` function, replicating its formatting and output behaviors.
* **libft:** A comprehensive custom C library that replicates various standard libc functions (like string manipulation, memory allocation, etc.) as well as additional utility functions.

## 🚀 How to Run
Since these projects are built in C, you will need a C compiler (like `gcc` or `clang`) to run them. Some projects include their own `Makefile` for easier compilation.

1. Clone the repository:
   ```bash
   git clone [https://github.com/ziqkimi308/collections-of-c-projects.git](https://github.com/ziqkimi308/collections-of-c-projects.git)
   ```
2. Navigate into the specific project folder you want to run.
3. If a `Makefile` is present, run `make` to compile:
   ```bash
    make
   ```

4. Execute the compiled output (usually `./a.out` or a specified binary name).
